1. Environment Setup

Open your terminal in the CDF-CHATBOT folder and run:

Create the virtual environment
python3 -m venv .venv

Activate the environment
source .venv/bin/activate

Install all required libraries
python -m pip install -r requirements.txt

2. Configuration

Create a file named .env in the root directory and add your OpenAI key:

OPENAI_API_KEY=your_sk_key_here

3. Build the Knowledge Base

Ensure your PDFs are inside the data/ folder. Then, run the indexing script to create the faiss_index/ folder:

python build_index.py

4. Launch the Bot

Run the Streamlit application:

python -m streamlit run app.py

Project Structure

.streamlit/: Contains UI theme configurations.

.venv/: Your Python virtual environment.

data/: Place all CDF PDF guides here.

faiss_index/: The local database generated by build_index.py.

.env: Stores your private API keys.

app.py: The main chatbot interface script.

build_index.py: Script to process PDFs into searchable data.

requirements.txt: List of all necessary Python packages.

Features

Source Citations: Every answer includes a "View Sources" dropdown showing the exact PDF filename and page number.

MMR Search: Uses Maximum Marginal Relevance to find diverse information across different pages, improving FAQ accuracy.

Anti-Hallucination: The bot is programmed to say "I don't know" if the answer is not present in the CDF guides.

Current Document Coverage

The bot is currently trained on:

Volunteer User Guide: Onboarding steps, Slack setup, and dashboard navigation. 

FAQ: Membership fees ($15/mo), late fees, waivers, and offboarding. 

EPM Admin Guide: Project monitoring and task tracking. 

Known Limitations

The bot cannot read scanned images; PDFs must have selectable text.
